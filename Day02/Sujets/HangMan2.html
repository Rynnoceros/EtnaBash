Instructions

Modify the hangman script for making the game asynchronous.

To achieve this, you need to store some datas in a file .hangmanstate in current directory.

When the game is over, delete that file.

Your script get the user proposition from STDIN.

To start a new game, the user must use option -n (short version) or --new (long version). When using this option, no input is required from the user.

How to store datas ?

Store datas in script format, so you can simply include that file in your script to read it (with .).

Example:

mystery="SCOOBIDOO"
found="_________"
triesleft=8
propositions=" Z I"
