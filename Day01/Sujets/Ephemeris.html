Instructions

Make a "ephemeris" script, it displays:

  • the day and the date
  • sunrise/sunset time
  • the ISS (International Space Station) location
  • the current distance between ISS and us !

It accepts an optional parameter: the API URL for ISS location. If not specified it will be "http://api.open-notify.org/iss-now.json".

Expected exact format result:

$> ./ephemeris.sh
Today we are Tuesday September  4, 2018.
Sunrise is expected at 5:12:13 AM and sunset at 6:26:23 PM.
The ISS is currently located at -30.1461, -50.7975: 10010km from us!
$>

Where I can find the data?
  • ISS location: http://api.open-notify.org/iss-now.json
  • Sunset/sunrise: https://sunrise-sunset.org/api
  • ETNA location is: 48.813875, 2.392521
  • PI value is: 3.14159265358
How can i calculate the distance between coordinates (lat1, lon1) and (lat2, lon2)?
earthRadiusKm = 6371;
dLat = (lat2 - lat1) * PI / 180;
dLon = (lon2 - lon1) * PI / 180;
l1 = (lat1) * PI / 180;
l2 = (lat2) * PI / 180;
a = sinus(dLat/2) * sinus(dLat/2) + sinus(dLon/2) * sinus(dLon/2) * cosine(l1) * cosine(l2);
result = 2 * atan2(sqrt(a), sqrt(1-a)) * eathRadiusKm;
